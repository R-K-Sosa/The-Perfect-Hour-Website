The 1-3-5 Checklist

Game Plan

  1. Consider the tasts that need to be done tomorrow, list out each of these tasks on a sheet of paper
    2.  
  2. Think about how long each task will take. Group the tasks by three size categories: Small, Medium, and Large
    3.  
  3. Which of these tasks are most important? Which of these must be done tomorrow and which can wait?
    4.  

Big Tasks

  1. Select one big task that you want to get done tomorrow and add it to your checklist. This should be something that will take at least an hour to complete in total and are of high importance.
    2.  
  2. Select the biggest item that you think you can feesably complete given your schedule tomorrow.
    3.  
  3. It may be helpful to break this task down into smaller parts if possible
    4.  

Medium Tasks

  1. select three medium sized tasks that you would like to complete tomorrow that will fit in your schedule. These should be tasks that will take you about a half hour to complete.
    2.  
  2. Consider assignments you may have for school. Which assignments have the shortest deadlines? Try to work on assignments that are coming due sooner, this way you won't be working on them at the last second.
    3.  
  3. These three items can also include any time consuming errands you may have to get out of the way.
    4.  

Small Tasks

  1. Select five small tasks that you want to get done tomorrow.
    2.  
  2. If you dont have 5 small things that you have to do, fill it in with things that you WANT to do!
    3.  
  3. These tasks can be small errands, cleaning, quick assignments, or simply leisurely activities!
    4.  
  4. It is important to make sure that while you are getting things done, you are still having a personally fulfilling day.
    5.  

Example

    Big Task:
  1. Complete a large textbook assignment for my investments class tonight
    Medium Tasks:
  1. Complete part of a presentation for my management class between classes
  2. Complete one small section of a paper for my finance class tonight
  3. Run to the store and buy groceries on my way home
    Small Tasks:
  1. Exercise
  2. Cook Dinner
  3. Organize my desk
  4. Watch an episode on Netflix after my work
  5. Start a new book before bed

Note: Make sure you can map out WHEN you will be doing each task!
